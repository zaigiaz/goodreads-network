* Overview

Curated Network of Goodreads titles that are clustered according to a different metric than
how Goodreads decides to choose what is in a Book Genre or not. Data from Google and Kaggle.
Slides were created with Latex-Beamer.

** Features

- Clustering method based off a computed metric for clustering book genres together
- Network Statistics and Comparisons
- Gephi Graph for visualization
- Comparison between a Random Network and Normal Clustered Network
- Final Interpretation of Results
