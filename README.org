* Overview

Curated Network of Goodreads titles that are clustered according to a different metric than
how Goodreads decides to choose what is in a Book Genre or not. Data from Google and Kaggle.
Slides were created with Latex-Beamer.

** Features

- Clustering method based off a computed metric for clustering book genres together
- Network Statistics and Comparisons
- Gephi Graph for visualization
- Comparison between a Random Network and Normal Clustered Network
- Final Interpretation of Results

** Datasets

- https://www.kaggle.com/datasets/ishikajohari/best-books-10k-multi-genre-data
- https://www.kaggle.com/datasets/mdhamani/goodreads-books-100k

** Usage

#+begin_src bash
cd src

# Create graph (saves to ../data/goodreads_network.gml)
python graph_builder.py

# With clustering coefficient (slow)
python graph_builder.py --clustering

# Load existing GML and compute stats + clustering
python graph_builder.py --from-gml

# Test different thresholds
python graph_builder.py --test-thresholds
#+end_src
